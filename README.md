📊 Pandas Data Cleaning Project

📖 Description

This is my first Pandas project using Python. In this project, I practiced basic data analysis and data cleaning with a CSV file.

🚀 Features

- Read CSV file using Pandas
- Display the dataset
- View first 5 rows ("head()")
- View last 5 rows ("tail()")
- Check dataset shape ("shape")
- Display dataset information ("info()")
- Find missing values ("isnull().sum()")
- Remove missing values ("dropna()")
- Check duplicate records ("duplicated().sum()")

🛠️ Technologies Used

- Python
- Pandas

📂 Project Files

- "main.py"
- "sahil_200.csv"

This project helped me understand:
